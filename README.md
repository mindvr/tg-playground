# tg-playground
Java Telegram Bot demos

## How to run

Register bot at [BotFather](https://t.me/BotFather). 
Set environment variables:

| Variable | Value        |
|----------|--------------|
| TG_NAME  | bot username |
| TG_TOKEN | bot token    |

Run as boot jar, docker TBD.
